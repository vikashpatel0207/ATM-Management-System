# ATM Management System (C++)

This is a simple ATM program made in C++.

## What this program can do
- Login using account number & PIN
- Withdraw money
- Change PIN
- Check balance
- Save data in file

## How to run
1. Compile
g++ main.cpp -o atm

2. Run
./atm

## accounts.txt format

AccountNumber PIN Name Balance

Example:
123456789 1234 Vikash 5000
